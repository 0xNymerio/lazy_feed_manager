Lazy Feed Manager

For lazy analysts :)

The Lazy Feed manage is a facilitator for you to start monitoring IOCs of IPs and Hashes in your environment and finally complete a step in The Pyramid of Pain. The intention is only to simplify and make it easier for you to integrate the feeds to your firewall, WAF, SIEM or whatever else you want to use to consume these materials. There are some vendors that have complications to feed their bases when we talk about IPV4 and IPV6, so I separated them for each endpoint and I did the same with the types of hashes algorithms. I hope that today you can monitor or correlate based on the feeds that are active, enjoy!

Endpoints to GET:

Log All IPs All Tor Ips All Hashes
Only IPv4 Only Tor IPv4 Only SHA256 Hashes
Python Log Only IPv6 Only Tor IPv6 Only SHA1 Hashes
- Only MD5 Hashes
